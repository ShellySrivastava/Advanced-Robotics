To use the package 'AR_week4_test'
1) Copy the package AR_week4_test to the source folder in catkin workspace (~/catkin_ws/src)
2) Open terminal at the catkin workspace (~/catkin_ws)
3) Compile the package using the following command
   catkin_make
4) Run roscore using the following command
   roscore
5) Use the launch file to start all the nodes, rqt_plot and rqt_graph. Use the command given below.
   roslaunch AR_week4_test cubic_traj_gen.launch
